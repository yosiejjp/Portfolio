My UX Skills

Ux Research

As a UX researcher, I specialize in understanding user needs and behaviours by conducting interviews, surveys, and usability tests. I create personas, and user journey maps, and employ empathy to ensure designs meet user expectations. Using A/B testing and analytics, I evaluate and refine designs to enhance user satisfaction and product success.

User Interface Design

As a UI designer, I create visually appealing and user-friendly interfaces. I use design tools to craft layouts, maintain consistency in elements, and ensure smooth navigation. Collaborating with UX researchers and developers, I focus on making modern and user-centred designs.

Web Development

As a web developer, I build interactive websites using HTML, CSS, and JavaScript. I focus on clean, efficient code for both front-end and back-end development. Collaborating with designers, I ensure visually appealing, user-friendly sites that work well across devices and browsers.
